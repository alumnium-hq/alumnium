# Calculator

URL: https://seleniumbase.io/apps/calculator.
Start in headless mode.
Don't use Alumnium planner.
Always save cache, even if tests are failed.

## Addition

1. Perform 2+2= operation.
2. Check that resuls is 4.

## Subtraction

1. Perform 9-5= operation.
2. Check that result is 4.

## Multiplication

1. Perform 6x7= operation.
2. Check that result is 42.

## Division

1. Perform 8/2= operation.
2. Check that result is 4.

## Decimal Addition

1. Perform 1.5+2.5= operation.
2. Check that result is 4.

## Chained Operations

1. Perform 2+3+5= operation.
2. Check that result is 10.

## Clear

1. Perform 7+3= operation.
2. Check that result is 10.
3. Press the CE (clear) button.
4. Check that result is 0.

## Division by Zero

1. Perform 5/0= operation.
2. Check that result indicates an error or is not a number.
