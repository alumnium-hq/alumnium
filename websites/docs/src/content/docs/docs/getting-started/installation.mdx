<LanguageContent lang="java">

Alumnium for Java requires JDK 21+ and is distributed via [Maven Central](https://central.sonatype.com/artifact/ai.alumnium/alumnium).

To install, add the library to your Gradle build along with a CLI artifact that matches your host platform.

```groovy title="build.gradle"
dependencies {
  testImplementation 'ai.alumnium:alumnium:0.21.0'
  testRuntimeOnly    'ai.alumnium:alumnium-cli-darwin-arm64:0.21.0'
}
```

Or, if you're using maven, add this to your pom.xml

```xml title="pom.xml"
<dependencies>
  <dependency>
    <groupId>ai.alumnium</groupId>
    <artifactId>alumnium</artifactId>
    <version>0.21.0</version>
    <scope>test</scope>
  </dependency>
  <dependency>
    <groupId>ai.alumnium</groupId>
    <artifactId>alumnium-cli-darwin-arm64</artifactId>
    <version>0.21.0</version>
    <scope>test</scope>
  </dependency>
</dependencies>
```

Replace `alumnium-cli-darwin-arm64` with the artifact matching your platform, or add multiple artifacts if you need to support multiple platforms. The available CLI artifacts are:

| OS      | Architecture | Artifact                     |
| ------- | ------------ | ---------------------------- |
| macOS   | arm64        | `alumnium-cli-darwin-arm64`  |
| macOS   | x64          | `alumnium-cli-darwin-x64`    |
| Linux   | arm64        | `alumnium-cli-linux-arm64`   |
| Linux   | x64          | `alumnium-cli-linux-x64`     |
| Windows | arm64        | `alumnium-cli-windows-arm64` |
| Windows | x64          | `alumnium-cli-windows-x64`   |

</LanguageContent>

<LanguageContent lang="python">

Alumnium requires Python 3+ and is distributed as a [PyPI package](https://pypi.org/project/alumnium/).

To install, run the following command:

```bash title="Installing dependencies…"
pip install alumnium
```

</LanguageContent>

<LanguageContent lang="typescript">

Alumnium for TypeScript requires Node.js 18+ and is distributed as an [NPM package](https://www.npmjs.com/package/alumnium).

To install, run one of the following commands:

<Tabs>
  <TabItem label="npm">

```bash title="Installing dependencies…"
npm install alumnium
```

  </TabItem>
  <TabItem label="pnpm">

```bash title="Installing dependencies…"
pnpm add alumnium
```

  </TabItem>
  <TabItem label="Yarn">

```bash title="Installing dependencies…"
yarn add alumnium
```

  </TabItem>
</Tabs>
</LanguageContent>
